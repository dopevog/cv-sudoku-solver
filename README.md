# Python Sudoku Solver Using OpenCv!
The porgram scans any image of a sudoku puzzle and solves it for you!
## Use It NOW
Clone the repo:
```
git clone https://github.com/dopevog/cv-sudoku-solver.git
cd cv-sudoku-solver
pip3 install opencv-python, numpy, tensorflow
python3 main.py
```
